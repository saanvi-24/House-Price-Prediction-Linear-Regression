# House Price Prediction using Linear Regression

## Overview

This project predicts California housing prices using a Linear Regression model built with Scikit-Learn.

The project demonstrates the complete Machine Learning workflow:

* Data Loading
* Exploratory Data Analysis (EDA)
* Data Preprocessing
* Model Training
* Prediction
* Model Evaluation

## Dataset

California Housing Dataset from Scikit-Learn

Features:

* Median Income
* House Age
* Average Rooms
* Average Bedrooms
* Population
* Occupancy
* Latitude
* Longitude

Target:

* Median House Value

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Jupyter Notebook

## Evaluation Metrics

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score
